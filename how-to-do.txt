Como adicionar nova regra no jogo: 
1- Criar a sua branch com o nome do que vai ser feito, de forma resumida:
 $ git checkout -b 'nome-da-branch-criada'
 
2- Edite o arquivo como desejar.

3- Abra o controle de branch.

4- Verificar o que esta certo e colocar em stage.

5- Comente o que esta subindo:
  $ git commit -m 'chore: comente aqui o que esta subindo'
  
6- Empurrar tudo que foi feito para a branch que voce criou no passo 1:
  $ git push origin nome-da-branch-criada
  
7- Criar um pull request no github.
